ecoinvent database for use in ARDA

Arda uses ecoinvent in .mat files. As ecoinvent is provided in ecospold2 format, the database must be parsed to .mat format prior to use in Arda. Both ecospold2matrix and Brightway (see example here for importing ecoinvent and exporting an SQLite3 database, which then can be converted to .mat) have utilities for this. Already exported mat files are kept on the X: drive at X:\indecol\DATA\ecoinvent\version_number.
