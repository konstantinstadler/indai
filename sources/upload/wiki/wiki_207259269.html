Industrial Ecology Digital Lab : e!Sankey

Created by Konstantin Stadler, last modified on 27.05.2021

Pro version of e!Sankey 5: https://www.ifu.com/en/e-sankey/online-shop/

This software is helpful to draw professional looking complex Sankey diagrams, without needing any coding or time-consuming editing. There is also an Excel link function for quick update of the data.

Usage

Use restricted for the MFA group, unless specified otherwise. We have a floating license for up to 2 concurrent users. For further questions and access, please contact Romain (romain.billy@ntnu.no).

Detailed use instructions available on the e!Sankey and data visualization channel on the MFA group's Team:
https://teams.microsoft.com/l/channel/19%3aa997e345c87f499a8767b961a4335c31%40thread.tacv2/e!Sankey%2520and%2520data%2520visualization?groupId=5d4c5168-c369-49f2-ae55-4213db9cbedb&tenantId=09a10672-822f-4467-a5ba-5bb375967c05

Administration

e!Sankey requires a Licence Programm running on a server (purchased in Jan 2021).

This is currently implemented on Francescas ArgGIS server (IEPT1398) at C:\Program Files\ifuFloatingLicenseService (requires admin rights).


After a reboot of the server, the service does not restart automatically. Instead, go to 'services' in the windows menu, search for ifufloatinglicenceservice , right click and start

Alternative Sankey software for MFA (Jan 2020)

A couple of free alternatives exist (mostly Python based), and they can still be useful for some use cases: 

  • https://github.com/ricklupton/floweaver: 
    This a python package that we have used in the past. Louis Monnier and Jennie Spruyt have used it for their master theses and Thomas Galea is using it for the MIND-P project. It works very well for simple systems and is easy to use even by someone not so familiar with Python. The results start getting a bit ugly for most complex systems, and it only works in Jupyter notebooks. It might still be interesting to use in some cases: 

    • You want a Sankey directly linked to the Python code 

    • You want to generate automatically a lot of Sankeys 

    • Your Sankeys are simple enough 

    • It does not need to be 100% professional looking 

  • https://plotly.com/python/sankey-diagram/ 
    The famous plotly visualization package in Python has a Sankey feature. It is comparable to floweaver, with some differences: 

    • It does not need Jupyter notebooks to run (you can use it with an IDE like Spyder or PyCharm. 

    • It is logical, nice and interactive, and you can reposition the flows, but it does not seem to work so well with backwards (e.g. recycling) loops. Then you have to specify the position of the different processes. 

    • Coupling this with Dash, you can quite easily create ince and responsive dynamic apps, with buttons that enable changing the year or other parameters of the model. This way, you can easily change the data displayed on the Sankey. It can be displayed on a website, etc., so it might be very useful for presentations or dissemination purposes. 

    • There is no convenient way to add labels for the values of the flows. When used in a browser, this is fine, because you can see them as tooltips when you hover over the flows, but it is annoying when you want to export the figure in a static format (like .png). 

    • It is widely used outside of the MFA community, which also explains why it is not really targeted to our use. 

  • http://www.visualisation.industrialecology.uni-freiburg.de/: 
    Developed by the indecol group in Freiburg (Stefan Pauliuk). Excel-based. Not very good looking at first try, requires further editing.

  • http://www.eco-data.fr/tools/sankey/start.php: 
    Louis Monnier used that for his project. It works with first uploading a csv file, then the flows and processes need to be manually adjusted, which has pros and cons: it is possible to make nice looking complex graphs, but it quickly becomes tedious. Updating the data afterwards is also more time consuming, but it is free, no installation or programming required, and works on Mac and Linux.

  • Additionally, check the overview here for further software http://www.sankey-diagrams.com/sankey-diagram-software/
