The Jupyter Project

The Jupyter Notebook is an open-source web application that allows you to create and share documents that contain
live code, equations, visualizations and explanatory text. Uses include: data cleaning and transformation, numerical simulation,
statistical modeling, machine learning and much more. http://jupyter.org/

If you have followed Anaconda installation Anaconda the Jupyter is already installed on your system.

We also have a jupyterhub running on our IndEcol Server.

IPython notebooks

At indecol we are going to use notebooks a lot and it might be convenient to open notebook just by clicking.
To do so open Anaconda prompt and execute:

pip install nbopen

Followed by:

  • Linux/BSD:

    python -m nbopen.install_xdg.

  • Windows:

    python -m nbopen.install_win

  • Mac: Clone the repository and run ./osx-install.sh

More info at: (https://github.com/takluyver/nbopen)
