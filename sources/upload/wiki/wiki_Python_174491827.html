Python is a programming language that lets you work quickly and integrate systems more effectively. There is huge community around the world and there is a chance your problem is already solved (using Python) so it it just a matter of googling around. Python has became language of choice for many areas (Software development, Art, Business, Education, Government, and yes Scientific! research too). Here at IndEcol we relay mainly on Pandas - powerful Python data analysis toolkit. For a list of useful Python packages (within in the standard library and external), click here.

Advantages of Python

Mostly compared to Matlab.

  1. Python can do the same things as Matlab and more: a multi purpose programming language with dedicated math libraries

  2. Open source: everyone can download a Python interpreter and run your code (matlab is expensive and proprietary)

  3. Python deals much better with complexity (debugging and maintaining complex code)

  4. Performance: xls parsing is much**2 faster in Python (and also available on Linux)

  5. Namespace: In python everything is a object and has an own namespace

  6. Ease of documentation (docstrings, automated documentation generation with sphinx)

  7. Object oriented (if you want to - it's not enforced as in Java)

  8. Matlab code can be executed within Python (http://stackoverflow.com/questions/2883189/calling-matlab-functions-from-python)

  9. File and functions are not the same (different names, for function/classes in one file, …)

  10. No semicolons at the end of the lines

Disadvantages

  1. Matrix handling is not as intuitive as in Matlab. Matlab is more to the point for linear algebra.

  2. No need for install extra packages and an IDE

Getting started

Installing

At IndEcol we usually use the latest stable Python version or one version below. We recommend, that you install the latest available python version for your system.

The easiest way to install python together with packages needed for scientific computation is through anaconda. Please follow the anaconda installing tutorial.

Why we do not recommend to install Python manually (in particular on Windows systems) because:

  • You need to set some environmental variables yourself

  • The Anaconda installation includes most Python data science packages you will need (Numpy, Pandas, …). Installing them afterwards can get complicated as some of them need to be compiled before installing.

  • It might break some of your environmental variables. In particular we observed problems with PATHEXT which gets set to .py, deleting .exe, .com, …

  • Anaconda includes the package manager conda which you can use to manage virtual environments (e.g. for different python versions and installed packages)

Using Python

You can use (i)python directly from the command line by starting “python” or “ipython” and write python programs in your preferred text editor. Alternatively, you can use an integrated development environment. One of them, spyder, is included in the anaconda distribution but behaves a little buggy from time to time.

In the Digital Lab we are using PyCharm, a professional python IDE by jetbrains. You can download a free community version following the link above. If you need the full version you can apply for a student/teacher account with your ntnu email address.
