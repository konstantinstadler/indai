Industrial Ecology Digital Lab : ArdaPy

Created by Unknown User (radekl), last modified on 11.11.2019

The next steps are based on assumption that either you already know how to work with python or you went through anaconda (all you need for python)

  1. Download ardapy-0.0.2.tar.gz

  2. open anaconda prompt (if you use anaconda otherwise feel free to use whatever you want), navigate to the file you have downloaded and run:

    pip install ardapy-0.0.2.tar.gz

  3. to create output for sankey diagram run script ardamat2json_sankey

    ardamat2json_sankey input.mat

    It might take few minutes to finish the script. The input.mat is the file created by arda matlab script.
    After the process is done the name_of_input_file.json file is created. You can then use this file at https://datavis.indecol.no/sankey, click on “choose file” (left upper corner) and select the created json file.
    On the web page use “Edit configuration” to style the sankey diagram.
    You can also specify output file name running script like

    ardamat2json_sankey input.mat -o my-output

    The my-ouptut.json file will be created then.
    Consider using verbose script if you want to see what is going on during running the script, the verbosity is use with -v or –verbose switch

    ardamat2json_sankey input.mat -v

  4. to create output for rosegraph diagram run script ardamat2json_rosegraph

    ardamat2json_rosegraph input.mat

    It might take few minutes to finish the script. The input.mat is the file created by arda matlab script.
    After the process is done the name_of_input_file.json file is created. You can then use this file at https://datavis.indecol.no/nestedpie, click on “upload data” (left upper corner) and select the created json file.
    You can also specify output file name running script like

    ardamat2json_rosegraph input.mat -o my-output

    The my-ouptut.json file will be created then.
    Consider using verbose script if you want to see what is going on during running the script, the verbosity is use with -v or –verbose switch

    ardamat2json_rosegraph input.mat -v

You can use -h (or –help) to see help doc for both script:

ardamat2json_sankey -h
ardamat2json_rosegraph --help
