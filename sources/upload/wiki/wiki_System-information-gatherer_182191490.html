Industrial Ecology Digital Lab : System information gatherer

Created by Konstantin Stadler, last modified on 13.01.2020



At the Digital Lab we keep a list of available hardware and used IP adresses/subspaces at IndEcol.

To help us with this, please use one of the scripts below, matching your operation system.


The script asks for your room-number (written on your door) and your IP-box number. For the IP box, you find the number on the white box connected which connects to your computer. If you are on a double box, the number before the slash refers to the left outlet, the other to the right one. Thus, if you have something like T419-X1034/1036 written on the box, use "T419-X0134" for the left one and "T419-X0136" for the right outlet.


Windows users can just use the script as is, for MacOS and Linux you need to download the script and make it executable with

chmod +x SCRIPTNAME

Then run the script in the terminal. If you encounter any problems, please contact the Digital Lab team.
