Industrial Ecology Digital Lab : Generate ssh-key for windows computers

Created by Unknown User (radekl), last modified on 11.11.2019

NB: This assumes you have already a version of Git installed.

In order to pull and push from a remote git repository your computer needs to be identified. One easy way of doing this is by making use of SSH. SSH creates a public/private key pair uniquely tied to your identity. Follow the following steps to generate an SSH-key for a windows computer.

Open up a command prompt from which git runs, e.g. Git Shell or Git Bash. Type the following command. (Include the “ around your email address)

 ssh-keygen -t rsa -C "your.email@ntnu.no"

This will generate a public/private rsa key pair.

Subsequently, you have to save the key in a directory. By default this will probably be /c/Users/[your_username]/.ssh/id_rsa. You can specify another path if you like. Otherwise, just hit enter.

Enter, and re-enter, a passphrase when prompted.

You will now have a public (.pub) and private ssh key located in /c/Users/[your_username]/.ssh
