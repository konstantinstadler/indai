Industrial Ecology Digital Lab : Clone remote repository to local pc

Created by Unknown User (radekl), last modified on 11.11.2019

Go to remote repository on e.g. gitlab, github, or bitbucket. Copy SSH to clipboard, e.g. git@gitlab.com:dlab-indecol/datavis.git

cd to the directory where you want to save the projects.

 cd 'C:\Users\[your_username]\[my_software_projects]\'

Clone subsequently the remote repository to your local drive

 git clone git@gitlab.com:dlab-indecol/datavis.git
