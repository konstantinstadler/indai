Coco - country converter

Coco used regular expression to find the best match of a specific country name to the known names. So far, it can convert all country names I encountered in the last four years. Coco provides output in several formats, including ISO2/3, EXIO, WIOD, UN numerical, FAO, â€¦

Coco is a python package, but can also be used in MATLAB and provides a command line interface.

e.g. in MATLAB:

pyversion folder/python.exe (replace folder with the directory where python.exe is located)
coco = py.country_converter.CountryConverter()
countries = {'The Swedish Kingdom', 'Norway is a Kingdom too', 'Peoples Republic of China', 'Republic of China'};
ISO2_pythontype = coco.convert(countries, pyargs('to', 'ISO2'));
ISO2_cellarray = cellfun(@char,cell(ISO2_pythontype),'UniformOutput',false);

e.g. command line (cmd in windows):

coco Norway

For further information see:
https://github.com/konstantinstadler/country_converter

To use coco, you need python (>=3.5) and matlab (>=2016a) available on your computer.

If you install with Anaconda everything should just work.

After you have that, you can install coco through:

pip install country_converter

A basic documentation you can find here:
https://github.com/konstantinstadler/country_converter

For some advanced usage in python see:
http://nbviewer.jupyter.org/github/konstantinstadler/country_converter/blob/master/doc/country_converter_examples.ipynb
