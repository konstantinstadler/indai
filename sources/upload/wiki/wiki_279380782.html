Industrial Ecology Digital Lab : matlab on linux server (installation guide)

Created by Ruslan Zhuravchak, last modified on 07.02.2023

This guide explains how to install/setup and use Matlab on the IEDL- maintained Linux servers

Prerequisites:

  1. an active Matlab license
  2. a Linux-based client computer (may not work if on Windows, use WSL in such case)

Installation procedure

NB: Do not use -i path_to_key option if the server login does not need an ssh key

  1. download the latest Matlab for Linux from Mathworks to your client machine (a zipped file shows up in the Downloads)
  2. create a temporary directory on the server, e.g: ssh my_username@server_ip -i path_to_key mkdir /home/my_username/tempdir
  3. copy the zipped file to the temporary directory at the server: scp -i path_to_key path_to_matlab/matlab_<VERSION>_glnxa64.zip my_username@server_ip:/home/my_username/tempdir
  4. ssh to the server with x-forwarding: ssh my_username@server_ip -i path_to_key -X
  5. change the permissions of the temporary directory: chmod -R 777 /home/my_username/tempdir
  6. navigate to the temporary directory: cd /home/my_username/tempdir
  7. unzip the archive: unzip matlab_<VERSION>_glnxa64.zip
  8. run the installer: ./install The window with the installation wizard shows up. Follow the instructions:
    1. enter the email and the password associated with your mathworks account
    2. agree with term and conditions
    3. select the appropriate license and confirm the user
    4. select the destination in your home directory, e.g. /home/my_username/mlab
    5. select the bare bones MATLAB and the necessary toolboxes, then proceed
    6. do not create the symbolic link
    7. confirm and finalize the installation
  9. you can now run Matlab by executing the command /home/my_username/mlab/bin/matlab ðŸŽ‰ Optionally, create an intuitive alias in your Bourne-compatible shell:
    1. open the configuration file with nano: nano /home/my_username/.profile
    2. at the end of the file, append the line that points to Matlab binaries: alias matlab='/home/my_username/mlab/bin/matlab'
    3. write the file (Ctrl+o) and close (Ctrl+x)
    4. source the config file : source /home/my_username/.profile or logout/login to the server.
    5. a shell-command matlab now starts Matlab
  10. remove the temporary directory from the server: rm rf /home/my_username/tempdir


To uninstall Matlab, remove the directory where Matlab was installed earlier, e.g: rm -rf /home/my_username/mlab and remove the alias from /home/my_username/.profile
