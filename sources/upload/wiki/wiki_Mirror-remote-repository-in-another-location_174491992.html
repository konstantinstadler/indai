Industrial Ecology Digital Lab : Mirror remote repository in another location

Created by Unknown User (radekl), last modified on 11.11.2019

Sometimes you want to be able to mirror a remote repository. For example, in the case you want to preserve work on a common (remote) repository in your own remote repository. Here is how to do it:

Create locally a bare mirrored clone of the repository to mirror:

 git clone --mirror https://github.com/exampleuser/repository-to-mirror.git

cd to the just cloned repo and set the push location to your mirror. NB. This often requires to create this empty repo to push to in github, bitbucket, or gitlab.

cd repository-to-mirror.git 
git remote set-url --push origin https://exampleuser@bitbucket.org/exampleuser/mirrored.git

Now you have to periodically monitor for changes in the original repository and push them to your mirror.

git fetch -p origin 
git push --mirror
