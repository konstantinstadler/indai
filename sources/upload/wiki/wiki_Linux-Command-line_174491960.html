There are several command lines in linux, we focus on bash here. 

However, you can use any of the installed shells your system (you find them with 

cat /etc/shells

). You can start any of these by entering the path given, and exit with exit().


Get to know the system

Get your username:

whoami


Get your hostname (and your ip)

hostname
hostname -i


More information about the the system

uname -a 


Main differences to the windows command line


  • The path separator is usually the slash ("/") compared to windows with the backslash ("\")
  • There is one root in linux ("/") whereas each drive is a separate root in windows (e.g. C:)
  • In linux you can fully customize the style of your prompt.
  • Parameters in commands are usually done with "-" compared to ("/") in windows (e.g. ls -l)

Getting help

COMMAND -h

man COMMAND
