Setting up the standard Industrial Ecology Work Environment

This page describes how to set-up the work environment on your PC. Please follow this installation guide as it leads to a consistent digital environment which allows us to reuse each others scripts and software.

Windows computer

  • Map the network drives as described here. It is important to use exactly the same mount points, otherwise scripts you write might not work on the machine of your colleagues.

  • Install Git on your PC

  • Install Python on your PC by using the Anaconda distribution

  • Check if you have an up-to-date matlab version installed, otherwise run the Software Center (win-key, than write “software center”) and install it. If you have admin rights on your PC, you can also install as described here

  • Finally, verify the correct installation and setup by testing your digital work environment as described here.

Linux

Install your preferred distribution. Currently, we have people working with Ubuntu and Manjaro (basically an Arch derivate). We can of course also help you with other distros, but we are most familiar with these two.

  • Map the network drives as described at the section Linux here

  • Git. This probably comes with your distro.

  • Python. You probably already have a system python included in your distro. Nevertheless, you should install a scientific python stack for development. We recommend to also use Anaconda for Linux. Follow the installation instructions here.

  • Check if you have an up-to-date matlab version installed, otherwise install as described here

  • Finally, verify the correct installation and setup by testing your digital work environment as described here.

macOS

  • Map the network drives using the guidelines for macOS

  • Use either of the options to install Git for macOS

  • Mac OS X, OS X and macOS have Python 2 preinstalled. Anaconda for macOS (recommended) comes with Python 3. Alternatively, Python 3 installer is available for downloading from the official website or using Homebrew. 

  • Install matlab for macOS from https://software.ntnu.no

  • Finally, verify the correct installation and setup by testing your digital work environment as described here.

BSD and other

Unfortunately, we do not have any experience with other operation systems. Probably most of the guide written for Linux should also work on the other unix-like OS. If you have any further information or helpful tips, please add them here.
