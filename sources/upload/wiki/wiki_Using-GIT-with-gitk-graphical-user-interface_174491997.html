Industrial Ecology Digital Lab : Using GIT with gitk graphical user interface

Created by Unknown User (radekl), last modified by Konstantin Stadler on 04.11.2020

Gitk is included in the standard git installation. It is graphical user interface to do the same what one can do using command line.

There is some tutorials on how to use it:

  • Short and good article: https://lostechies.com/joshuaflanagan/2010/09/03/use-gitk-to-understand-git/

  • 4min youtube tutorial: https://www.youtube.com/watch?v=yfivNnk0_Gg


If you like to have a more advanced GUI interface we recommend GitAhead.
