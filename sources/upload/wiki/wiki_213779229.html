USING THE TERMINAL IN VSCODE ON THE WINSERVER:

You might encounter a problem when trying to open a terminal in vscode on the vmserver.
This can be solved by changing the setting of "Terminal.Integrated.Default Profile: Windows"

Go to settings (gear symbol, then settings) , then type "terminal.integrated.windows" in the search bar.
Then you find "Default Profile" Windows below, select one of the available shells for your terminal (command prompt probably, or whatever you prefer)

Powershell probably does not work (security settings on the server), but git bash should also work
