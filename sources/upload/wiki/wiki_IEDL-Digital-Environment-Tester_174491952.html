Testing the digital work environment

After you have installed git, python and matlab as described here, verify if all things work together:

Windows PC

If you are on a MS Windows machine, run the file

X:\indecol\Software\iedl_checks\software_check\iedl_checks.exe

If the last line of the test output says “OK”, you are good to go. If it says “WARNING - NOT ALL EXPECTED PROGRAMS FOUND”, please double check if you followed the installation guide exactly. When you still encounter problems, please open a command line(Windows), change the directory by typing these two lines:

X: 
cd indecol\Software\iedl_checks\software_check

and run

iedl_checks -f

and email the output of the extended tests to the Digital Lab.

Other OS - installing the system checks program manually

If you are using a Mac or Linux system, you need to install the system checks program:

  1. Open a command line (Windows) or terminal (Linux).

  2. Install the IndEcol Digital Environment Tester by copy-pasting the following line into the command prompt window:

    pip install git+https://gitlab.com/dlab-indecol/iedl_checks.git --upgrade

    Press enter.

  3. Close the command line window and open another one.

  4. Copy - paste

    iedl_checks

    into the command line and start the tests with enter (the tests might take a couple of seconds).

  5. If the last line of the test output says “OK”, you are good to go. If it says “WARNING - NOT ALL EXPECTED PROGRAMS FOUND”, please double check if you followed the installation guide exactly. When you still encounter problems, please run

    iedl_checks -f

    and email the output of the extended tests to the Digital Lab.
