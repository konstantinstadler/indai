Industrial Ecology Digital Lab : Copy existing ssh-key to associate with Gitlab account

Created by Unknown User (radekl), last modified on 11.11.2019

For windows users.

In order to pull and push from and to remote repositories, you will often have to provide the public part of your ssh-key.

In order to do so, open a command terminal and, if necessary, change the directory to your home folder. (/c/Users/[your_username]) In windows powershell the following will do:

 cd ~

Assuming the name of the public key you want to copy is id_rsa, type :

 type .ssh\id_rsa.pub | clip

The public ssh key is now copied to your clipboard and you can paste it where needed.

(NB) the above copying command works in Powershell. In Git Bash

 cat .ssh/id_rsa.pub

and select and copy the terminal output to paste where needed
