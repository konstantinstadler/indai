Industrial Ecology Digital Lab : Atom

Created by Ruslan Zhuravchak, last modified on 25.03.2021

Atom is a lightweight text editor for writing the code in multiple languages developed for Windows, linux and macOS. Bare bones of Atom features syntax highlightIng, code autocompletion, file manager and the GUI for version control with git. With numerous packages available on demand, it can be turned into an all-inclusive IDE for interactive analytics, development and debugging of the research software. This guide explains how to install atom and some useful packages for running with python:

1) Download/install Atom. Use the online manual to guide through the installation process.

2) Install hydrogen - a package for interactive programming. The easiest way is to run "apm install hydrogen" in your terminal. Set up the language mapping: go to Preferences → Packages → Hydrogen → Language Mappings and make sure that atom's "python" (on the left) points to the environmental path of your python installation (on the right), e.g. { "python": "python3" } or { "python": "\C:\Programs\Anaconda3\envs\test_env\python" }. Test you installation by selecting one or more line(s) of code and running them (using the shortcut Shift+Enter). If necessary, setup IPython kernel as prompted. An example below demonstrates a working Atom+Hydrogen setup for the interactive programming.

3) Install any additional packages that suit your needs. You may want to consider script,  minimap, autocomplete-python, python-debugger, python-autopep8, linter-flake8.
