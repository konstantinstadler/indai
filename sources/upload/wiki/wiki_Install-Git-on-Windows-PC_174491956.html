Industrial Ecology Digital Lab : Install Git on Windows PC

Created by Unknown User (radekl), last modified by Konstantin Stadler on 04.11.2020

Git is the standard version control system we are using at Industrial Ecology. We highly recommend that you use git to track changes in your (source code) files. Even if you are not using git for version control, you need to install git for installation of some packages provided by the Digital Lab.

  • Download git here. This download includes the standard command line git version plus some simple GUI tools.
  •  After downloading, double click the install file. Click next to accept the GNU GPL licence.
  •  In the Select components windows, you can probably leave all settings to the standard values. Double-check, however:

    1. that both subcategories of “Windows Explorer Integration” (“Git Bash Here” and “Git GUI Here”) are selected.

    2. that “Associate .git* config…” with the text editor is selected

    3. that “Associate .sh files to be run with Bash” is selected

  •  In the next step select your preferred text editor.
  •  Adjusting your PATH environment. You can either choose “Use Git from the Windows command prompt” or “Use Git and optional Unix tools”. Both are fine, we recommend the latter (although it comes with a warning).
  •  Select “Use the OpenSSL library”.
  •  Configuring the line ending conversion. Follow the recommended setting for your system (Windows, Unix)
  •  Configuring the terminal emulator: This is up to you, we recommend the default setting (Use MinTTy)
  •  Configuring extra option. Leave the default setting (“Enable file system caching” and “Enable Git Credential Manager”)
  •  Vote out of any experimental options

After installation open a command line shell and enter “git”. You should get a help message. If not, please contact the Digital Lab.

You can start to use git.


If you want to use git with a graphical user interface, we recommend to use GitAhead (available for Windows, Mac and Linux).
