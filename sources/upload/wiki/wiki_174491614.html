Industrial Ecology Digital Lab : Digital Tools for IndEcol Research/Teaching

Created by Unknown User (radekl), last modified by Konstantin Stadler on 04.10.2021

Software tools and databases specifically developed for Industrial Ecology research. This list contains software which is ready for research/production.

Internally developed software and databases

  •  Arda - a in-house developed LCA calculator
  •  ArdaPy - Arda Python utility to use with Arda results
  •  Themis - a hybrid LCA/MRIO system for energy systems
  •  Pymrio - a in-house developed MRIO calculator
  •  K_MRIO_Folder - MRIOs with data, metadata, and concordance matrices
  •  Coco - country converter for python, matlab and command line
  •  ais_query - python helper function to query maritime MongoDB AIS database OBSOLETE - USE maridata
  •  maridata - python tool to query MariData MongoDB database (smart team project)
  •  EXIOBASE
  •  JupyterHub - https://jupyter.indecol.no
  •  IEDL Digital Environment Tester - test for installed programs, hardware and environmental variables
  •  IAM_tools
  •  DataVis
  • Ecoinvent Impact Browser - online web tool for Johan Berg Pettersen course - only form NTNU network
  • JupyterHub for teaching- setting up and using the JupyterHub for teaching classes


Externally developed tools and databases

  •  Brightway - a LCA calculator
  •  sankey-truthstudio - sankey plot used in Aluminium MFA
  •  e!Sankey - sanky tool for the MFA/Daniels group
  •  WIOD
  •  Ecoinvent
  •  SimaPro
  •  IndEcol Open Source Software and Database Dashboard
