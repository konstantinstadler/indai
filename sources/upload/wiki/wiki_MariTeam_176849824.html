Data

Data comes form multiple sources and are stored in MongoDB database running on Indecol Server at maridata db name.

The maridata schema:

  • messagges - collection of row dynamic part of AIS data. (source: http://www.kystverket.no/AIS via Sintef) Multiple data cleaning procesessed:

    1. drop messages without mmsi or x or y or timestamp

    2. remove duplicates (same mmsi and x and y and timestamp)

    3. removing messages where -180 > x > 180 and -90 > y > 90

    4. add loc to use mongodb geoindex 2dsphere

  • messages_completed - Result of completer script which completes missing routes of origin AIS messages. the messages are then updated with weather and eca data.

  • specs - collection of static part of AIS data. (source: http://www.kystverket.no/AIS via Sintef)

  • specs_log - log information during parsing of AIS data per each line of data file.

  • logs - log information during parsing of AIS data per one data file.

  • sea_web - collection of processed seaweb data with All documents with mmsi=null or missing mmsi field are removed. (source: http://maritime.ihs.com/)

  • ports - collection of all ports. (source: https://msi.nga.mil/NGAPortal/MSI.portal?_nfpb=true&_pageLabel=msi_portal_page_62&pubCode=0015)

  • weather - weather historical data based on http://apps.ecmwf.int/datasets/data/interim-full-daily/levtype=sfc/. The weather collection fields are:

    • time: hours since 1.1.1900

    • unxtimestamp : unit: unix time

    • timestamp : string in format: '%Y%m%d %H:%M:%S'

    • lat : unit: degrees north, range [-90,90]

    • lon : unit: degrees east, range [0,359.25]

    • v10 : units: , long_name: 10 metre V wind component

    • u10 : units: , long_name: 10 metre U wind component

    • wind_speed : units , formula:

    • wind_direction: degree true, formula: 270 - (math.atan2(x['u10'] / x['wind_speed'], x['v10'] / x['wind_speed']) * 180 / math.pi) % 360

    • wave_height : units: m, long_name: Significant height of combined wind waves and swell

    • wave_direction : units: Degree true, long_name: Mean wave direction

  • eca - Emission Control Areas. Each filed is one restricted area with name and loc: 2dsphere indexed geolocation
