Phase 1 - Basic Python, setup environment and Jupyter notebook

Here, we show you the steps to get started.
There are two alternative ways of setting stuff up:

Alternative 1: Through Anaconda Prompt

Step 1: Digital work environment set up. now WHAT?

So, you've set up your indecol_digital_work_environment and you're happpy. Now what?

Step 2: Connecting to the interactive notebook we follow in the course

a) Open your Anaconda prompt
b) Press the green clone or download button here: https://github.com/jakevdp/WhirlwindTourOfPython
c) Copy the URL and type the following command in Anaconda Prompt:

git clone [insert copied url here]

d) Type the following commands in Anaconda Prompt:

cd WhirlwindTourofPython
jupyter notebook
Step 3: Have fun with the tutorials and play around with/ruin the code

The website now pops up in your internet browswer of choice. Now you can click the different tutorials and change the code however you like.

Phase 2 - NumPy
