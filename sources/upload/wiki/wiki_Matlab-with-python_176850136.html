Using matlab from Python

based on https://se.mathworks.com/help/matlab/matlab_external/install-matlab-engine-api-for-python-in-nondefault-locations.html

Windows

Open Anaconda promt and
If you do not have write permission to build the engine (which probably you don't) in the MATLAB folder, use a nondefault folder, builddir.

cd "matlabroot\extern\engines\python"
python setup.py build --build-base="builddir" install

Most probably the matlabroot will be something like: C:\Program Files\MATLAB\R201xx

Mac or Linux

cd "matlabroot/extern/engines/python"
python setup.py install

Testing

To test if the installation of Matlab API was successful open python interpreter and run:

import matlab.engine 
eng = matlab.engine.start_matlab() 
t = eng.gcd(100.0,80.0,nargout=3) 
print(t)

This should print: (20.0, 1.0, -1.0)

you can see more at: https://se.mathworks.com/help/matlab/matlab_external/call-matlab-functions-from-python.html
