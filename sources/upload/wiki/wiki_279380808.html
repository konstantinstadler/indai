Industrial Ecology Digital Lab : IndEcol1(2) windows server

Created by Ruslan Zhuravchak, last modified on 31.01.2023

The Windows server for ArcGIS and other Windows GUI programs.

The main purpose of this server is for ArcGIS tasks which are too big for local desktops and which benefit from a GPU. Python/Matlab scripts should still mostly run on our Linux servers (there will also be a new one with GPU available soon).

Access

Go to https://desktop.ntnu.no .
There you have the choice to use the server within the browser or to install a VMware client locally. Login credentials for both are your NTNU username and password.

If you choose the within-browser option you will then find a button with "IndEcol Desktop 1" and 2.

Usage


The Windows server is managed through VMware which allows 2x5 users to be logged in simultaneously (5 on Indecol1, 5 on Indecol2).

There is no time-out for the sessions to allow for long-running calculations.
This also means that we need some logout discipline: when you are done, logout (not just disconnect or close the browser window) from the server to allow others to use the server. Just closing the browser tab is not enough!

For the usual use case - login and use the server. If you have a particular heavy job you can announce it on the TEAMS channel infra-VMwareServer - if you see such announcements please logout and free all resources.

- Network-Storage: The X-drive(forsking Eksperimentell) is mounted to X: for all users.
- Local-Storage: Mapped on W:. There is no backup for this drive(!) - this should only be used for intermediate results. Please make a directory with your username or project name (if you share among users) for storing your data within your Prof Group sub directory.

Updates/Downtime


Every second Thursday in a month at 16:00 the server will be updated (security patches and such). IT should inform active users but plan for this on your side as well. If you are logged in and not active you will logged out.

Hardware Details/Storage


PowerEdge R750 with 2x Xeon Gold 6342 (24 Cores). 512 GB RAM, NVIDIA AMPERE A40 GPU with 48 GB RAM, 6.5 TB SSD.
The server is split into 2 instances, so you should see half of these resources when you login.
