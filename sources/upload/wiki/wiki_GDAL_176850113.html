Industrial Ecology Digital Lab : GDAL

Created by Unknown User (radekl), last modified on 11.11.2019

For GDAL we are using docker from GDAL2 github: https://github.com/GeographicaGS/Docker-GDAL2.

The usage:

  • SSH to Indecol server

  • Run

    docker pull geographica/gdal2:latest

    (change latest to any required version as mentioned on github)

  • Run docker. Probably you want to use host system directory with data. Lets assume you have data at /shared/someuser and you want to mount it to /home directory on docker image, then run command:

    docker run -v /shared/someuser/:/home -t -i geographica/gdal2 /bin/bash

  • cd to home directory

    cd /home

    where your data are

Here is other github with GDAL docker images (a bit older versions) of GDAL https://github.com/geo-data/gdal-docker

  • SSH to Indecol server

  • run

    git clone https://github.com/geo-data/gdal-docker.git

  • if you want to work with your data located on Indecol server (lets say in /shared/yourname/gdal_data) run:

    docker run -v /shared/radekl:/data -t -i geodata/gdal /bin/bash
